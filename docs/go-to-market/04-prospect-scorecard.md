# Prospect Qualification Scorecard

## Scoring Model
- Score each criterion from 1 to 5.
- Weighted total >= 70: strong pilot candidate.
- Weighted total 55 to 69: nurture and re-evaluate.
- Weighted total < 55: deprioritize.

## Qualification Checklist
- Clear operational pain mapped to one of Nexora pilot use cases.
- Named executive sponsor and day-to-day operator owner.
- Data access and environment access available.
- Security review path defined.
- Pilot timeline accepted.

## Pilot Readiness Questions
- What is the current onboarding time for a new edge device?
- What is the average incident response time today?
- Which compliance controls must be demonstrated in pilot?
- What internal systems must be integrated during pilot?
- What is the success definition for day 30 and day 60?

## Next Action Matrix
- High score + high urgency: schedule technical discovery within 5 business days.
- High score + low urgency: run value workshop and set target kickoff window.
- Low score + high urgency: narrow scope to one low-risk use case.
- Low score + low urgency: keep in nurture list.
